💡 Usage Examples

JavaScript - Fetch All Cards

const res = await fetch('https://arthurb2l.github.io/japan-tcg-price-guide/data/brain-cache.json');
const data = await res.json();
const allCards = Object.values(data.sets).flat();
console.log(`${allCards.length} cards loaded`);

JavaScript - Search by Name

const cards = Object.values(data.sets).flat();
const charizards = cards.filter(c => 
  c.name.toLowerCase().includes('charizard')
);
console.log(charizards.map(c => `${c.name} - €${c.price?.eur}`));

JavaScript - Top 10 Most Valuable

const top10 = Object.values(data.sets).flat()
  .filter(c => c.price?.eur)
  .sort((a,b) => b.price.eur - a.price.eur)
  .slice(0, 10);
console.table(top10.map(c => ({name: c.name, price: c.price.eur})));

Python

import requests
data = requests.get('https://arthurb2l.github.io/japan-tcg-price-guide/data/brain-cache.json').json()
cards = [card for cards in data['sets'].values() for card in cards]
print(f"{len(cards)} cards loaded")
